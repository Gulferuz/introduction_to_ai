#  Introduction to AI: Линейная регрессия

Этот репозиторий содержит практическую работу по основам машинного обучения. Мы исследуем зависимость дистанции бега от количества съеденных картофелин, проходя путь от ручных вычислений до использования библиотек ИИ.

## Описание проекта

В проекте реализованы три подхода к решению задачи:
1.  **Ручной расчет на NumPy**: Вычисление коэффициентов наклона ($w_1$) и смещения ($w_0$) через средние значения и суммы.
2.  **Визуализация на Matplotlib**: Построение диаграммы рассеяния (`scatter`) и линии тренда для анализа данных.
3.  **Машинное обучение со Scikit-learn**: Использование профессионального класса `LinearRegression` для обучения модели и прогнозирования.



## 🛠 Технологический стек
* **Python 3.x**
* **NumPy** — математические вычисления и работа с массивами.
* **Matplotlib** — построение графиков и визуализация.
* **Scikit-learn** — построение модели линейной регрессии.
* **JupyterLab** — интерактивная среда для разработки и запуска кода.

##  Как запустить проект

1) Подготовка папки
Скачайте файлы проекта (или склонируйте репозиторий) в новую папку и откройте её в VS Code
2) Создание виртуального окружения 
```bash
python -m venv .venv 
```
3) Активация окружения
Активируйте среду, чтобы компьютер начал использовать Python из папки .venv
Для Windows:
```bash
.venv\Scripts\activate
```
Для Mac/Linux:
```bash
source .venv/bin/activate
```

Важно: В начале строки терминала должно появиться сокращение (.venv).

4) Установка библиотек
```bash
pip install -r requirements.txt
```
