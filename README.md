#  Introduction to AI



##  Как запустить проект

1) Подготовка папки
Скачайте файлы проекта (или склонируйте репозиторий) в новую папку и откройте её в VS Code
2) Создание виртуального окружения 
```bash
python -m venv .venv 
```
3) Активация окружения
Активируйте среду, чтобы компьютер начал использовать Python из папки .venv


Для Windows:
```bash
.venv\Scripts\activate
```

Для Mac/Linux:
```bash
source .venv/bin/activate
```

Важно: В начале строки терминала должно появиться сокращение (.venv).


4) Установка библиотек
```bash
pip install -r requirements.txt
```
